\subsection{Detailed Description}
The X.\-509 module provides X.\-509 support which includes\-:
\begin{DoxyItemize}
\item X.\-509 certificate (C\-R\-T) reading (see {\ttfamily x509parse\-\_\-crt()} and {\ttfamily x509parse\-\_\-crtfile()}).
\item X.\-509 certificate revocation list (C\-R\-L) reading (see {\ttfamily x509parse\-\_\-crl()} and{\ttfamily x509parse\-\_\-crlfile()}).
\item X.\-509 (R\-S\-A and E\-C\-C) private key reading (see {\ttfamily x509parse\-\_\-key()} and {\ttfamily x509parse\-\_\-keyfile()}).
\item X.\-509 certificate signature verification (see {\ttfamily x509parse\-\_\-verify()})
\item X.\-509 certificate writing and certificate request writing (see {\ttfamily mbedtls\-\_\-x509write\-\_\-crt\-\_\-der()} and {\ttfamily mbedtls\-\_\-x509write\-\_\-csr\-\_\-der()}).
\end{DoxyItemize}

This module can be used to build a certificate authority (C\-A) chain and verify its signature. It is also used to generate Certificate Signing Requests and X509 certificates just as a C\-A would do.
